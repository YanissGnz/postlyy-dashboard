# Postlyy

Postlyy is a social media management platform that allows users to manage their social media accounts in one place. Users can schedule posts, view analytics, and engage with their audience all from one platform.

## Technologies Used

- Next.js
- React
- Tailwind CSS
- Radix UI
- Shadcn UI
- DigitalOcean
- Redux Toolkit
